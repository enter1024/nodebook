
CSS中所有对隐藏的处理
=================================================


1.
=============================================================
 display: none           display: block ;  显示的意思 
 visibility: hidden;       visibility: visible  显示的意思



2.
=================================================
 display  隐藏元素并且不占位置
 visibility:hidden 隐藏元素并且占有位置   停职留心
 

3.
=================================================
 overflow:hidden;   隐藏超出的部分。
